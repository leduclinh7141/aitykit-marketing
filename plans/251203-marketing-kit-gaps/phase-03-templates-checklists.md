# Phase 3: Marketing Templates & Checklists

## Overview
Real-world marketers need actionable checklists and templates, not just concepts.

## 3.1 Daily Checklists

### Social Media Daily Checklist
```markdown
## Morning (9 AM)
- [ ] Check overnight mentions/DMs
- [ ] Respond to urgent messages
- [ ] Review today's scheduled posts
- [ ] Share 1-2 industry articles

## Midday (12 PM)
- [ ] Engage with comments (all platforms)
- [ ] Monitor brand mentions
- [ ] Check trending topics for opportunities

## Afternoon (3 PM)
- [ ] Review post performance
- [ ] Schedule tomorrow's content if gaps
- [ ] Respond to remaining comments
- [ ] Update content ideas list

## End of Day (5 PM)
- [ ] Final engagement sweep
- [ ] Note tomorrow's priorities
- [ ] Log any issues/wins
```

### Email Daily Checklist
```markdown
- [ ] Check delivery rates
- [ ] Monitor spam complaints (<0.1%)
- [ ] Review unsubscribe rate
- [ ] Check sequence performance
- [ ] Respond to replies
```

## 3.2 Weekly Checklists

### SEO Weekly Checklist
```markdown
## Rankings
- [ ] Check target keyword positions
- [ ] Note ranking changes (+/-)
- [ ] Identify quick win opportunities

## Technical
- [ ] Check Google Search Console errors
- [ ] Review crawl stats
- [ ] Fix any 404s

## Content
- [ ] Update underperforming pages
- [ ] Add internal links to new content
- [ ] Check for content decay

## Backlinks
- [ ] Monitor new backlinks
- [ ] Check for toxic links
- [ ] Outreach for link opportunities
```

### Content Weekly Checklist
```markdown
- [ ] Review content calendar
- [ ] Assign content tasks
- [ ] Review drafts in pipeline
- [ ] Approve scheduled content
- [ ] Update editorial calendar
- [ ] Brainstorm new topics
```

## 3.4 Campaign Launch Checklist
```markdown
## Pre-Launch (1 Week Before)
- [ ] Campaign strategy approved
- [ ] Creative assets finalized
- [ ] Landing pages live and tested
- [ ] Tracking pixels installed
- [ ] UTM parameters created
- [ ] Email sequences ready
- [ ] Social posts scheduled
- [ ] Ad copy approved
- [ ] Budget confirmed

## Launch Day
- [ ] All campaigns live
- [ ] Initial performance check
- [ ] Team notified
- [ ] Social posts published
- [ ] Email blast sent

## Post-Launch (24 Hours)
- [ ] Performance review
- [ ] Initial optimizations
- [ ] Bug fixes if needed
- [ ] Stakeholder update
```
