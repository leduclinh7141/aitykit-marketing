## Script Writing Tips

### For Voiceover/Talking Head:
- Write how you speak (conversational)
- Use contractions (don't, can't, won't)
- Short sentences (easier to deliver)
- Mark pauses [PAUSE]
- Note emphasis **like this**
- Read aloud before filming

### For Text-Only/Silent:
- Short, punchy text
- One idea per screen
- Easy to read quickly
- High contrast text
- Sync with visual changes

### Pacing:
- **Fast:** Social media, short attention spans
- **Medium:** Educational, tutorial
- **Slow:** Emotional, storytelling, luxury

### Engagement Techniques:
- Ask questions
- Use pattern interrupts
- Show relatable problems
- Provide clear value
- Create curiosity gaps
- Use humor (when appropriate)
