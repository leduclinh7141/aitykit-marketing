# On-Page SEO

Checklist for optimizing individual pages for search.

## Title Tag Optimization

**Best Practices**
- 50-60 characters max
- Primary keyword near beginning
- Include brand at end (if room)
- Unique for each page

**Formula**
`Primary Keyword - Secondary Keyword | Brand Name`

**Examples**
- "Email Marketing Guide: 15 Tips to Boost Open Rates | AityKit"
- "Best CRM Software 2024: Top 10 Compared | Reviews"

## Meta Description

**Best Practices**
- 150-160 characters
- Include primary keyword
- Add call-to-action
- Compelling value proposition

**Formula**
`[Benefit] + [Primary keyword context] + [CTA]`

## Header Hierarchy

```
H1: Primary Topic (keyword)
├── H2: Main Section 1
│   ├── H3: Subsection
│   └── H3: Subsection
├── H2: Main Section 2
└── H2: Main Section 3
```

## Content Optimization

### Keyword Placement
- [ ] In title tag
- [ ] In meta description
- [ ] In H1 tag
- [ ] In first 100 words
- [ ] In at least one H2
- [ ] In image alt text
- [ ] In URL slug

### Content Quality
- [ ] Comprehensive coverage
- [ ] Original insights
- [ ] Current information
- [ ] Proper formatting
- [ ] Visual elements
- [ ] External source links

## Internal Linking

- Link to relevant pages
- Use descriptive anchor text
- Prioritize important pages
- Fix orphan pages
- Maintain reasonable link count

## Image Optimization

- Descriptive file names
- Alt text with keywords
- Compressed file sizes
- Proper dimensions
- Lazy loading enabled
