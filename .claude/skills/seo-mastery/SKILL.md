# SEO Mastery

Search engine optimization strategies and tactics for organic growth.

## Activation

Use when optimizing content for search, conducting keyword research, or performing SEO audits.

## Core Concepts

### Search Intent Types

| Intent | User Goal | Content Type |
|--------|-----------|--------------|
| Informational | Learn something | How-to, guides, definitions |
| Navigational | Find specific site | Brand pages, product pages |
| Commercial | Research before buying | Comparisons, reviews, lists |
| Transactional | Complete purchase | Product pages, pricing |

### On-Page SEO Elements

**Title Tag** (50-60 characters)
- Primary keyword near front
- Compelling and click-worthy
- Unique per page

**Meta Description** (150-160 characters)
- Include primary keyword
- Call-to-action
- Unique value proposition

**Header Structure**
- H1: One per page, contains keyword
- H2-H3: Logical hierarchy
- Include keyword variations

**Content Optimization**
- Keyword in first 100 words
- Natural keyword density (1-2%)
- Semantic variations throughout
- Internal linking to related content

### Technical SEO

**Core Web Vitals**
- LCP (Largest Contentful Paint): <2.5s
- FID (First Input Delay): <100ms
- CLS (Cumulative Layout Shift): <0.1

**Crawlability**
- XML sitemap submission
- Clean robots.txt
- Fix broken links
- Proper redirects

**Indexability**
- Canonical tags
- No duplicate content
- Proper noindex usage
