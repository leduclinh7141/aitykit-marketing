# TikTok Ads

Advertising on TikTok.

## Ad Formats

### In-Feed Ads
- Native format
- Up to 60 seconds
- Full-screen vertical
- Swipe up CTA

### TopView
- First ad seen
- Premium placement
- Up to 60 seconds
- High impact, high cost

### Branded Hashtag Challenge
- User participation
- Viral potential
- High engagement
- Significant investment

### Branded Effects
- Custom filters/stickers
- Interactive experiences
- Brand awareness focus

### Spark Ads
- Boost organic posts
- Native appearance
- Higher engagement
- Lower production cost

## Creative Best Practices

### Format
- 9:16 vertical
- 15-30 seconds optimal
- Native TikTok style
- Sound on design

### Content
- Hook in first 3 seconds
- Show product in action
- Use trending sounds
- Include captions
- Clear CTA

### Style
- Authentic, not polished
- User-generated aesthetic
- Humor and entertainment
- Trend participation

## TikTok Pixel

### Events to Track
- PageView
- ViewContent
- AddToCart
- CompletePayment
- SubmitForm

### Best Practices
- Use Events Manager
- Set up CAPI
- Test event firing
- Use UTM parameters
