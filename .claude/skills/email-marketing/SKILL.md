# Email Marketing

Email campaign strategy, automation, and optimization.

## Activation

Use when creating email sequences, improving deliverability, or designing email automation workflows.

## Core Concepts

### Email Types

| Type | Purpose | Timing |
|------|---------|--------|
| Welcome | Onboard new subscribers | Immediate |
| Nurture | Build trust over time | Drip sequence |
| Promotional | Drive sales/signups | Campaign-based |
| Transactional | Confirm actions | Triggered |
| Re-engagement | Win back inactive | Behavior-triggered |

### Key Metrics

| Metric | Good | Excellent |
|--------|------|-----------|
| Open Rate | 15-25% | 25%+ |
| Click Rate | 2-5% | 5%+ |
| Unsubscribe | <0.5% | <0.2% |
| Bounce Rate | <2% | <0.5% |

### Email Anatomy

```
From: Name <email@domain.com>
Subject: [Hook + Benefit]
Preview: [Extends subject curiosity]

[Personalized greeting]
[Hook - address pain/desire]
[Value delivery]
[Social proof - optional]
[CTA button]
[P.S. - additional hook]
```
