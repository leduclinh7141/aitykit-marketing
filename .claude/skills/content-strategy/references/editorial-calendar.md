# Editorial Calendar

Planning and managing content production.

## Calendar Structure

### Essential Fields
| Field | Purpose |
|-------|---------|
| Publish Date | When it goes live |
| Content Type | Blog, video, social |
| Title/Topic | What it's about |
| Keyword | SEO target |
| Funnel Stage | TOFU/MOFU/BOFU |
| Owner | Who's responsible |
| Status | Draft/Review/Published |
| Channel | Where it's distributed |

### Optional Fields
- Content pillar
- Campaign association
- Promotion plan
- Performance metrics
- Notes/links

## Planning Cadence

### Annual
- Major campaigns
- Seasonal themes
- Product launches
- Industry events

### Quarterly
- Content pillar focus
- Campaign alignment
- Resource allocation
- Goal setting

### Monthly
- Specific topics
- Production schedule
- Distribution plan
- Performance review

### Weekly
- Final assignments
- Review queue
- Publishing schedule
- Promotion tasks

## Content Mix Planning

### By Format (example)
| Format | Frequency |
|--------|-----------|
| Blog posts | 2-4/week |
| Videos | 1-2/week |
| Podcasts | 1/week |
| Ebooks | 1/quarter |
| Webinars | 1/month |

### By Pillar
- Distribute evenly across pillars
- Weight by strategic priority
- Seasonal adjustments

## Production Workflow

1. **Ideation** (Week -4)
2. **Brief/Outline** (Week -3)
3. **First Draft** (Week -2)
4. **Review/Edit** (Week -1)
5. **Final Polish** (Week -1)
6. **Publish** (Week 0)
7. **Promote** (Week 0-2)
