# Analytics & Attribution

Performance measurement and attribution modeling.

## Activation

Use when setting up tracking, analyzing performance, or building attribution models.

## Core Concepts

### Analytics Framework

**Dimensions**: What you're measuring by
- Channel, campaign, source
- Device, geography, time
- Audience segment

**Metrics**: What you're measuring
- Traffic, engagement, conversion
- Revenue, cost, ROI

### Key Reports

| Report | Questions Answered |
|--------|-------------------|
| Acquisition | Where do visitors come from? |
| Behavior | What do they do on site? |
| Conversion | Do they complete goals? |
| Attribution | What drove the conversion? |

### Attribution Models

| Model | Credit Distribution |
|-------|-------------------|
| Last Click | 100% to final touchpoint |
| First Click | 100% to first touchpoint |
| Linear | Equal across all |
| Time Decay | More to recent touches |
| Position | 40/20/40 first-mid-last |
| Data-Driven | ML-based distribution |
