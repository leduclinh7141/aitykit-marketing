description: Stage, commit and push all code in the current branch
---
Stage all changed files, create a meaningful commit based on the changes, and push to remote repository.

**Workflow:**
1. Run `git status` to see all changes
2. Run `git diff` to review changes
3. Run `git log -3` to see recent commit message style
4. Stage relevant files with `git add`
5. Create commit with descriptive message following conventional commits
6. Push to remote with `git push`
7. Verify with `git status`
