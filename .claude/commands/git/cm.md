Stage all changed files and create a commit following conventional commit format.

**Workflow:**
1. Run `git status` to see all changes
2. Run `git diff` to review staged and unstaged changes
3. Run `git log -3` to see recent commit message style
4. Stage relevant files with `git add`
5. Create commit with descriptive message
6. Run `git status` to verify

**IMPORTANT: DO NOT push the changes to remote repository**
