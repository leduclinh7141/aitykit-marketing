# Marketing Performance Report

## Objective
Generate a comprehensive marketing performance report for the specified timeframe and channels.

## Workflow

1. **Data Collection**
   - Traffic metrics
   - Conversion data
   - Engagement metrics
   - Revenue attribution

2. **KPI Calculation**
   - Visitor metrics
   - Lead generation
   - Conversion rates
   - Cost metrics
   - Revenue metrics

3. **Benchmark Comparison**
   - vs Previous period
   - vs Same period last year
   - vs Industry benchmarks
   - vs Targets

4. **Trend Analysis**
   - Growth trends
   - Seasonal patterns
   - Channel performance trends
   - Content performance trends

5. **Visualization**
   - Key metrics dashboard
   - Trend charts
   - Channel breakdown
   - Funnel visualization

6. **Executive Summary**
   - Key wins
   - Areas of concern
   - Recommendations
   - Next steps

## Output
- Complete performance report
- Executive summary
- Recommendations
