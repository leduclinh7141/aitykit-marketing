# Viral Content Creation

## Objective
Create content with high viral potential for the specified topic and platform.

## Workflow

1. **Viral Pattern Analysis**
   - Current trending formats
   - Successful content patterns
   - Platform algorithm preferences
   - Audience sharing triggers

2. **Format Identification**
   - Native formats (Reels, TikToks, Stories)
   - Trending templates
   - Audio/sound trends
   - Visual styles

3. **Viral Framework Application**
   - Emotional triggers (awe, humor, inspiration)
   - Shareability factors
   - Hook optimization
   - CTA for engagement

4. **Content Variant Creation**
   - Multiple hook options
   - Different angles
   - A/B test versions
   - Platform adaptations

5. **Algorithm Optimization**
   - Hashtag strategy
   - Caption optimization
   - Posting timing
   - Engagement baiting (ethical)

## Viral Triggers

| Trigger | Emotion | Example |
|---------|---------|---------|
| Awe | Wonder | Transformation reveals |
| Humor | Joy | Relatable situations |
| Controversy | Curiosity | Hot takes |
| Utility | Gratitude | Life hacks, tips |
| Identity | Belonging | "Tag someone who..." |

## Output
- Viral content brief
- Multiple content versions
- Platform-specific adaptations
