# Landing Page Copy

## Objective
Create high-converting landing page copy for the specified offer and audience.

## Workflow

1. **Audience Analysis**
   - Pain points
   - Desired outcomes
   - Objections
   - Decision criteria

2. **Value Proposition**
   - Core benefit
   - Unique differentiator
   - Proof points
   - Risk reversal

3. **Copy Structure**
   - Hero section
   - Problem agitation
   - Solution presentation
   - Social proof
   - FAQ/objections
   - Final CTA

4. **Conversion Optimization**
   - Clear CTA hierarchy
   - Trust signals
   - Urgency elements
   - Form optimization

## Landing Page Structure

```markdown
## Hero Section
**Headline:** [Big promise/benefit]
**Subheadline:** [How you deliver]
**CTA:** [Primary action]
**Social proof:** [Quick credibility]

## Problem Section
[Agitate the pain point]

## Solution Section
[Present your solution]

### Feature 1
[Benefit-focused description]

### Feature 2
[Benefit-focused description]

## Social Proof
- [Testimonial 1]
- [Testimonial 2]
- [Stats/logos]

## FAQ
**Q:** [Common question]
**A:** [Answer + pivot to benefit]

## Final CTA
[Urgency + clear action]
```

## Output
- Complete landing page copy
- Section-by-section breakdown
- CTA variations
