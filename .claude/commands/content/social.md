# Social Content Creation

## Objective
Create platform-optimized social media content for the specified topic.

## Workflow

1. **Platform Analysis**
   - Format requirements
   - Character limits
   - Best practices
   - Algorithm preferences

2. **Content Adaptation**
   - Platform-native format
   - Visual specifications
   - Copy style
   - Hashtag strategy

3. **Engagement Optimization**
   - Hook optimization
   - CTA placement
   - Engagement triggers
   - Reply/comment strategy

4. **Variant Creation**
   - Multiple hooks
   - Different angles
   - A/B test versions
   - Carousel/series options

## Platform Specifications

| Platform | Format | Copy Length | Hashtags |
|----------|--------|-------------|----------|
| LinkedIn | Long-form, document | 1-3k chars | 3-5 |
| Twitter/X | Thread, visual | 280 chars/tweet | 1-2 |
| Instagram | Reel, carousel | 125-150 chars | 20-30 |
| Facebook | Video, link post | 40-80 chars | 1-3 |
| TikTok | Short video | 150 chars | 3-5 |

## Content Template

```markdown
## [Platform] Post

**Hook:** [First line that stops scroll]

**Body:**
[Main content]

**CTA:** [What you want them to do]

**Hashtags:** #tag1 #tag2 #tag3

**Visual:** [Image/video description]
```

## Output
- Platform-specific content versions
- Visual specifications
- Posting recommendations
