# Email Content Creation

## Objective
Create compelling email copy for the specified type and objective.

## Email Types

| Type | Purpose | Key Elements |
|------|---------|--------------|
| Welcome | Onboard | Value prop, next steps |
| Nurture | Educate | Pain points, solutions |
| Promotional | Convert | Offer, urgency, CTA |
| Newsletter | Engage | Value, updates, CTAs |
| Transactional | Inform | Clear info, next steps |
| Re-engagement | Win back | New value, incentive |

## Workflow

1. **Context Analysis**
   - Email objective
   - Audience segment
   - Funnel stage
   - Prior engagement

2. **Subject Line Creation**
   - Multiple variations
   - Curiosity/benefit focus
   - Personalization tokens
   - A/B test options

3. **Email Body**
   - Compelling opener
   - Value delivery
   - Social proof
   - Clear CTA

4. **Sequence Design** (if applicable)
   - Email flow
   - Timing
   - Triggers
   - Exit conditions

## Email Template

```markdown
## Email: [Name]

**Subject:** [Subject line]
**Preview:** [Preview text]

---

[Personalized greeting],

[Hook - address pain/opportunity]

[Body - deliver value]

[Social proof - optional]

[CTA button text]

[Sign-off]

---

**Send timing:** [When]
**Segment:** [Who]
**Goal:** [What action]
```

## Output
- Complete email copy
- Subject line variations
- Sequence map (if applicable)
