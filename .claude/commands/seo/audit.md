# SEO Audit

## Objective
Perform a comprehensive SEO audit and generate prioritized recommendations.

## Workflow

1. **Technical SEO Analysis**
   - Site speed assessment
   - Mobile-friendliness check
   - Crawlability analysis
   - Schema markup review
   - Core Web Vitals evaluation

2. **On-Page Analysis**
   - Title tag optimization
   - Meta description review
   - Heading structure analysis
   - Content quality assessment
   - Internal linking audit

3. **Content Analysis**
   - Thin content identification
   - Duplicate content check
   - Content gap analysis
   - Keyword mapping review

4. **Off-Page Analysis**
   - Backlink profile overview
   - Domain authority assessment
   - Toxic link identification

5. **Competitive Positioning**
   - SERP position tracking
   - Share of voice analysis
   - Competitor comparison

## Output
- Comprehensive SEO audit report
- Prioritized action items by impact/effort
- Technical specifications for fixes
