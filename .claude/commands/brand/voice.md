# Brand Voice Guidelines

## Objective
Create comprehensive brand voice guidelines for consistent communication.

## Workflow

1. **Brand Positioning Analysis**
   - Review brand mission/vision
   - Understand value proposition
   - Identify differentiators
   - Analyze target audience

2. **Voice Attribute Definition**
   - Core personality traits (3-5)
   - What we are / What we're not
   - Voice spectrum (formal â†” casual)
   - Emotional qualities

3. **Tone Variations**
   - By channel (social, email, web)
   - By situation (support, sales, education)
   - By audience segment
   - By content type

4. **Do's and Don'ts**
   - Language to use
   - Language to avoid
   - Grammar preferences
   - Punctuation rules

5. **Example Copy**
   - Headlines
   - CTAs
   - Error messages
   - Social posts
   - Email signatures

## Voice Framework

```markdown
## Our Voice Is:
1. [Attribute 1] - [Description]
2. [Attribute 2] - [Description]
3. [Attribute 3] - [Description]

## We Sound Like:
- [Example]
- [Example]

## We Never Sound Like:
- [Example]
- [Example]
```

## Output
- Brand voice guidelines document
- Example copy library
- Quick reference card
