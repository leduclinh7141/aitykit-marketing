# Brand Asset Management

## Objective
Inventory, organize, and manage brand assets.

## Actions

### Inventory
- Catalog existing assets
- Identify gaps
- Check version currency

### Organize
- Create folder structure
- Apply naming conventions
- Tag with metadata

### Audit
- Review asset quality
- Check brand compliance
- Identify outdated assets

## Workflow

1. **Asset Inventory**
   - Logos (all formats/sizes)
   - Color palette files
   - Typography files
   - Photography/imagery
   - Templates
   - Icons and graphics

2. **Categorization**
   - By asset type
   - By use case
   - By channel
   - By format

3. **Metadata Tagging**
   - File names
   - Description
   - Usage rights
   - Version info
   - Expiration dates

4. **Usage Guidelines**
   - Approved uses
   - Restrictions
   - Modification rules
   - Attribution requirements

5. **Version Control**
   - Current versions
   - Archive structure
   - Update process
   - Distribution method

## Asset Structure

```
brand-assets/
├── logos/
│   ├── primary/
│   ├── secondary/
│   └── variations/
├── colors/
├── typography/
├── imagery/
├── templates/
└── icons/
```

## Output
- Asset inventory spreadsheet
- Organized folder structure
- Asset usage guide
