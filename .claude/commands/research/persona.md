# Buyer Persona Creation

## Objective
Create detailed buyer persona for the specified product or segment.

## Workflow

1. **Data Collection**
   - Customer interviews
   - Survey data
   - Analytics insights
   - Sales feedback

2. **Pattern Identification**
   - Common characteristics
   - Shared challenges
   - Similar goals
   - Behavior patterns

3. **Demographics**
   - Age range
   - Job title/role
   - Company size
   - Industry
   - Location

4. **Psychographics**
   - Goals and motivations
   - Pain points
   - Values and beliefs
   - Decision criteria

5. **Behavior Documentation**
   - Content consumption
   - Channel preferences
   - Buying process
   - Influences
