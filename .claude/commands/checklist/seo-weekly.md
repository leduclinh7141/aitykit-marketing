## Output Format
```markdown
# Weekly SEO Checklist: Week of [Date]

## Monday: Rankings & Performance Review

### Search Console Check (20 min)
- [ ] Review Performance report (last 7 days vs previous)
  - [ ] Clicks trend: ___
  - [ ] Impressions trend: ___
  - [ ] CTR trend: ___
  - [ ] Position trend: ___
- [ ] Check for new queries driving traffic
- [ ] Identify declining keywords (action needed)
- [ ] Review top pages performance
- [ ] Check mobile vs desktop performance

### Rankings Tracker (15 min)
- [ ] Export current rankings snapshot
- [ ] Compare to last week
- [ ] Note significant changes (+/- 5 positions):
  - [ ] Winners: ___
  - [ ] Losers: ___
- [ ] Investigate ranking drops
- [ ] Document ranking wins

### Analytics Review (15 min)
- [ ] Organic traffic trend
- [ ] Bounce rate changes
- [ ] Top landing pages
- [ ] Conversion rate from organic
- [ ] New vs returning organic visitors

---

## Tuesday: Technical SEO Health

### Crawl & Indexing (20 min)
- [ ] Check Search Console Coverage report
- [ ] Review crawl errors (404s, 500s)
- [ ] Check pages excluded from indexing
- [ ] Verify sitemap is up to date
- [ ] Submit any new pages for indexing

### Site Health (25 min)
- [ ] Run site audit (Screaming Frog/Ahrefs/Semrush)
- [ ] Check for new broken links
- [ ] Review redirect chains
- [ ] Check for duplicate content issues
- [ ] Verify canonical tags
- [ ] Check robots.txt for issues

### Core Web Vitals (15 min)
- [ ] Review PageSpeed Insights for key pages
- [ ] Check LCP (Largest Contentful Paint) < 2.5s
- [ ] Check FID (First Input Delay) < 100ms
- [ ] Check CLS (Cumulative Layout Shift) < 0.1
- [ ] Log any pages needing optimization

---

## Wednesday: Content Optimization

### Content Audit (30 min)
- [ ] Identify underperforming content (low traffic, high impressions)
- [ ] Find content with declining rankings
- [ ] List content needing updates (> 6 months old)
- [ ] Check for thin content pages
- [ ] Review content gaps vs competitors

### On-Page Optimization (30 min)
- [ ] Optimize 1-2 existing pages:
  - [ ] Update title tags (include target keyword)
  - [ ] Refresh meta descriptions (improve CTR)
  - [ ] Add/update header tags (H2, H3)
  - [ ] Improve internal linking
  - [ ] Update outdated information
  - [ ] Add relevant images with alt text
  - [ ] Improve content depth/quality

### Featured Snippet Opportunities (15 min)
- [ ] Identify keywords ranking positions 2-10
- [ ] Check if SERP has featured snippets
- [ ] Optimize content format for snippet capture
- [ ] Add FAQ schema where appropriate

---

## Thursday: Link Building & Authority

### Backlink Monitoring (20 min)
- [ ] Check for new backlinks gained
- [ ] Review lost backlinks
- [ ] Identify toxic backlinks (disavow if needed)
- [ ] Monitor competitor backlink gains
- [ ] Note link building opportunities

### Outreach Pipeline (30 min)
- [ ] Research 5-10 link prospects
- [ ] Send 3-5 outreach emails
- [ ] Follow up on pending outreach
- [ ] Document responses and wins
- [ ] Update link building tracker

### Internal Linking (15 min)
- [ ] Add internal links to new content
- [ ] Link from high-authority pages to target pages
- [ ] Fix orphaned pages (no internal links)
- [ ] Review anchor text distribution

---

## Friday: Competitor & Keyword Analysis

### Competitor Monitoring (25 min)
- [ ] Check competitor rankings changes
- [ ] Review competitor new content
- [ ] Identify competitor keyword wins
- [ ] Note competitor backlink gains
- [ ] Document competitive gaps/opportunities

### Keyword Research (20 min)
- [ ] Research new keyword opportunities
- [ ] Analyze keyword difficulty
- [ ] Check search intent alignment
- [ ] Prioritize keywords for next content
- [ ] Update keyword tracking spreadsheet

### Weekly Reporting (15 min)
- [ ] Compile weekly metrics
- [ ] Document wins and challenges
- [ ] Set priorities for next week
- [ ] Update team/client on progress
```
