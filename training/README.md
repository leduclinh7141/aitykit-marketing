# AityKit Marketing Training

Interactive learning modules for mastering AI-powered marketing workflows with Claude Code.

## Overview

This training program teaches you to use aitykit-marketing through hands-on practice. Unlike video courses, you learn by doing real marketing work inside Claude Code.

**Learning Method:** Interactive lessons taught by Claude
**Practice Project:** Markit agency working for client Planerio
**Time Investment:** 4-6 hours total
**Prerequisite:** Claude Code installed

## Training vs Production

### Training Mode

**Purpose:** Learn through guided lessons
**Commands:** `/training:*`
**Project:** Practice with Markit/Planerio
**Focus:** Understanding concepts and building skills

### Production Mode

**Purpose:** Real marketing work for your business
**Commands:** Main commands (`/campaign:*`, `/content:*`, etc.)
**Project:** Your actual marketing projects
**Focus:** Delivering results efficiently

### Transition Strategy

1. **Complete training modules** - Learn the system
2. **Apply to small project** - Test on real work
3. **Build your templates** - Customize for your brand
4. **Scale up** - Use for all marketing campaigns
